stiff_1d_se

PURPOSE

STIFF_1D_SE Assembles 1D global SEM matrix associated to ((phi_j)', (phi_i)')

SYNOPSIS

function [A]=stiff_1d_se(npdx,ne,nov,wx,dx,jacx);

DESCRIPTION 

 STIFF_1D_SE Assembles 1D global SEM matrix associated to  ((phi_j)', (phi_i)')


    [A]=stiff_1d_se(npdx,ne,nov,wx,dx,jacx); produces the matrix
        A_{ij}=((phi_j)', (phi_i)') of size
        (noe,noe) where noe=nov(npdx,ne) is the number of d.o.f.

 Input : npdx = number of nodes in every element
         ne = number of spectral elements
         nov = local -global map, previously generated by cosnov1d
         wx = npdx LGL weigths in [-1,1],
            (produced by calling [x,w]=xwlgl(npdx))
            (npdx = number of nodes, =n+1, if n=polynomial degree)
         dx = first derivative LGL matrix (produced by calling d=derlgl(x,npdx))
         jacx =  jacobian of the map F:[-1,1]---->[xa_ie,b_ie]

 Output: A = matrix (noe,noe) 


 Reference: CHQZ2 = C. Canuto, M.Y. Hussaini, A. Quarteroni, T.A. Zang,
                    "Spectral Methods. Fundamentals in Single Domains"
                    Springer Verlag, Berlin Heidelberg New York, 2006.

CROSS-REFERENCE INFORMATION

This function calls:
  • stiff_1d_sp STIFF_1D_SP Computes local stiffness matrix for 1D problem
This function is called by:
  • ellprecofem_1d ELLPRECOFEM_1D FEM-preconditioned SEM-appx of 1D second order b.v.p.
  • lap_1d LAP_1D SEM appx of the 1D b.v.p. -nu u''+gam u=f
  • matrices_1d MATRICES_1D Assembles SEM stiffness and mass matrices for 1D b.v.p.
