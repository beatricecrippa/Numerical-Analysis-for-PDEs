plot_2dmodal

PURPOSE

PLOT_2DMODAL plots 2D modal boundary adapted polynomials.

SYNOPSIS

function plot_2dmodal

DESCRIPTION 

 PLOT_2DMODAL  plots 2D modal boundary adapted polynomials.
             Formula (2.3.31), pag. 82, CHQZ2 (tensorial product),
             produces part of Fig. 2.13, pag. 100 CHQZ2

     plot_2dmodal   % no input, no output

 Reference: CHQZ2 = C. Canuto, M.Y. Hussaini, A. Quarteroni, T.A. Zang,
                    "Spectral Methods. Fundamentals in Single Domains"
                    Springer Verlag, Berlin Heidelberg New York, 2006.

CROSS-REFERENCE INFORMATION

This function calls:
  • pol_legendre POL_LEGENDRE recursive construction of Legendre basis function
  • pol_modal POL_MODAL recursive construction of modal basis function
