plot_modal

PURPOSE

PLOT_MODAL plots 1D modal boundary-adapted polynomials

function plot_modal

DESCRIPTION 

 PLOT_MODAL  plots 1D modal boundary-adapted polynomials 
             Formula (2.3.31), pag. 82, CHQZ2,
